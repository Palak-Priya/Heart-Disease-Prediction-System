# ❤️ Heart Disease Prediction System

## 📌 Description
This project predicts heart disease using machine learning and provides health suggestions.

## 🚀 Features
- Heart disease prediction
- Health feedback system
- Previous reports (history)
- PDF report download
- User login system

## 🛠️ Technologies Used

### 💻 Frontend
- HTML5
- CSS3

### ⚙️ Backend
- Python
- Flask Framework

### 🧠 Machine Learning
- Jupyter Notebook (Anaconda Navigator)
- Pandas (data cleaning & preprocessing)
- NumPy (numerical operations)
- Scikit-learn (model training & prediction)
- Pickle (model saving & loading)

### 🗄️ Database
- SQLite (for storing user history & reports)

### 📊 Additional Libraries
- ReportLab (for PDF report generation)

### 🔧 Tools & Platforms
- Anaconda Navigator
- Git & GitHub (version control)
- VS Code (code editor)

## ⚠️ Note
Currently using a dummy model for demonstration. Real ML model can be integrated.

## ▶️ How to Run Project

1. Clone the repository:
   git clone https://github.com/your-username/heart-disease-prediction-system.git

2. Go to project folder:
   cd heart-disease-prediction-system

3. Install required libraries (Flask, Pandas, NumPy, etc.)

4. Run the Flask app:
   python app.py

5. Open browser and visit:
   http://127.0.0.1:5000
